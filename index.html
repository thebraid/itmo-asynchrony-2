Promises

Создаем Promise


							
								const promise = new Promise((resolve, reject) => {
								  // функция-исполнитель (executor)
								});

resolve и reject - коллбэки, предоставляемые самим JS

Свойства Promise

  • state (состояние)
    • pending (ожидание)
    • fulfilled (выполнено успешно)
    • rejected (выполнено с ошибкой)
  • result (результат)

Схема

Потребители: then, catch и finally

Объект Promise позволяет подписаться на его состояние

							
								const promise = new Promise(...);

								promise.then(func1, func2)
  • func1 - функция выполнится, когда промис переходит в состояние «fulfilled», и получает результат

  • func2 - функция выполнится, когда промис переходит в состояние «rejected», и получает ошибку

Примеры

Успешное завершение:

							
								const promise = new Promise((resolve, reject) => {
								  setTimeout(() => resolve("ready"), 1000);
								});

								promise.then(
								  result => alert(result), // выведет "ready" через 1 cек
								  error => alert(error) // не будет запущена
								);

Обработка ошибки:

							
								const promise = new Promise((resolve, reject) => {
								  setTimeout(() => reject(new Error("oops")), 1000);
								});

								promise.then(
								  result => alert(result), // не будет запущена
								  error => alert(error) // выведет "Error: oops"
								);

Пример с обработкой только ошибки:

							
								const promise = new Promise((resolve, reject) => {
								  setTimeout(() => reject(new Error("oops")), 1000);
								});


								promise.then(null, error => alert(error));
								// ===
								promise.catch(error => alert(error));

Пример с finally:

							
								const promise = new Promise((resolve, reject) => {
								  setTimeout(() => resolve("ready"), 1000);
								});


								promise
									.then(...);
									.catch(...);
									// выполнится независимо от наличия ошибки
									.finally(() => /*убрать индикатор загрузки*/);

Поменяем порядок finally:

							
								const promise = new Promise((resolve, reject) => {
								  setTimeout(() => resolve("ready"), 1000);
								});


								promise
									// нет аргументов
									.finally(() => console.log(1));
									.then(...);
									.catch(...);

Промисификация

Промисификация

Это преобразовании функции, которая принимает callback, в функцию, возвращающую promise

Промисификация

							
								const loadData = (callback) => {
									const response = ... // загружаем данные;

									callback(response);
								};

								loadData(data => {
									console.log(data);
								})

Промисификация

							
								const loadData = () => {
									return new Promise(resolve => {
										const response = ... // загружаем данные;

										resolve(response);
									})
								};

								loadData()
									.then(response => console.log(response))

Универсальная функция промисификации

							
								import { readFile } from 'node:fs';

								readFile('/etc/passwd', (err, data) => {
									if (err) throw err;
									console.log(data);
								});

Универсальная функция промисификации

							
								const promisify = (fn) => {
									return (...args) => {
										return new Promise((resolve, reject) => {
											fn(...args, (error, data) => {
												if (error) {
													reject(error);
												}

												resolve(data);
											})
										})
									}
								}

Универсальная функция промисификации

							
								import { readFile } from 'node:fs';

								const readFilePromisify = promisify(readFile);

								readFilePromisify('/etc/passwd')

									.catch(error => console.log(error))

Promise API

Promise.all

объединяет несколько промисов и выполняет их параллельно

							
								Promise.all([
									promise1, // возвращает 1
									promise2, // возвращает 2
								])
									.then(([result1, result2]) => {
										console.log(result1); // 1
										console.log(result2); // 2
									})

Promise.allSettled

как Promise.all, но возвращает объект со статусом и результатом промиса для каждого промиса

							
								Promise.allSettled([
									promise1, // возвращает 1
									promise2, // возвращает 2
								])
									.then(([result1, result2]) => {
										console.log(result1); // {status: "fulfilled", value: 1}
										console.log(result2); // {status: "fulfilled", value: 2}
									})

Promise.race

возвращает первый завершенный промис (вне зависимости завершился успешно или нет)

							
								Promise.race([
									promise1,
									promise2,
								])
									.then((result) => {
										console.log(result);
									})

Promise.any

возвращает первый успешно завершившийся промис

							
								Promise.any([
									promise1,
									promise2,
								])
									.then((result) => {
										console.log(result);
									})

Promise.resolve

Создает Promise в состоянии fulfilled с установленным значением

							
								Promise.resolve(5)

Promise.reject

Создает Promise в состоянии rejected

Где кроется проблема?

Вот бы отменить предыдущий запрос

Проблема - у Promise нет встроенных механизмов отмены

Решение: fetch + AbortController = ❤️

							
								let controller = new AbortController();
  • controller - объект, который содержит 1 метод abort() и 1 свойство signal
  • При вызове abort() генерируется событие "abort" на объекте controller.signal
  • Свойство controller.signal.aborted становится = true

Рассмотрим пример с AbortController (без fetch)

							
								const controller = new AbortController();
								const signal = controller.signal;

								signal.addEventListener('abort', () => console.log("1"));

								controller.abort();

								console.log(signal.aborted); // true

Возвращаемся к fetch

							
								const controller = new AbortController();

								fetch(url, {
								  	signal: controller.signal
								});

								controller.abort(); // запрос будет отменен

Финальная реализация

							
								let controller = undefined;

								const getSuggest = (text) => {
									if (controller) {
										controller.abort();
									}

									controller = new AbortController();

									return fetch('...', {
										signal: controller.signal
									})
										.then(response => response.json());
								}

								getSuggest('тор');
								getSuggest('торт');
